# Staff User Guide

## Logging In
1. Go to the app URL
2. Enter your email and password
3. You'll be taken to the Log Inventory page

## Submitting a Log

### Deduction (Used/Refill)
Use this when you open a new pack or use ingredients.

1. Select **Used / Refill**
2. Use the **category tabs** (All, Powder, Liquid, Addon) to filter ingredients
3. **Search** by typing the ingredient name
4. Click the ingredient to select it
5. Enter the **quantity** (number of packs/items)
6. Click **Submit Log**

### New Delivery
Use this when new stock arrives.

1. Select **New Delivery**
2. Select the ingredient
3. Enter the quantity delivered
4. Click **Submit Log**

## Tips
- Your branch is automatically detected — no need to select it
- Use category tabs to quickly find ingredients
- Search works as you type — no need to press enter
- You'll see a green success toast when the log is submitted
